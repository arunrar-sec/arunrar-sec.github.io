/ suid_finder

A focused script designed to identify SUID and SGID binaries on a Linux filesystem, which are common targets for local privilege escalation. 


#!/bin/bash
# Focused SUID/SGID Finder

echo "[+] Finding all SUID binaries..."
find / -perm -4000 -type f -exec ls -al {} \; 2>/dev/null

echo -e "\n[+] Finding all SGID binaries..."
find / -perm -2000 -type f -exec ls -al {} \; 2>/dev/null

By identifying these binaries, an attacker or auditor can then check against databases like GTFOBins to see if any of them can be exploited for elevated shell access.
