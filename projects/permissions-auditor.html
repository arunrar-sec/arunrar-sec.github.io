/ permissions_auditor

A bash script that audits common Linux File Hierarchy Standard (FHS) permissions and highlights insecure configurations such as world-writable directories and sensitive files. 


#!/bin/bash
# Linux Permissions Auditor

echo "[*] Auditing world-writable directories..."
find / -xdev -type d \( -perm -0002 -a ! -perm -1000 \) -print 2>/dev/null

echo "[*] Auditing world-writable files..."
find / -xdev -type f -perm -0002 -print 2>/dev/null

echo "[*] Checking for sensitive file permissions..."
ls -l /etc/shadow /etc/passwd /etc/group

This auditor is a fundamental tool for system hardening and identifying potential privilege escalation vectors on a Linux system.
