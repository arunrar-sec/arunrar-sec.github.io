Phase 1: Linux Control (Current)

Shell & Internals

+

Mastering the shell, file systems, permissions, and bash scripting.

1.1 Introduction to Linux

Linux File Hierarchy Standard (FHS)

Understanding directory structures, core directories like /etc, /var, /bin, and navigating the system efficiently.

1.2 Identity & Access

User and Permission Management

Managing users, groups, working with chmod, chown, sudoers, and understanding the significance of the root user.

1.3 Data Manipulation

Text Processing Tools

Advanced usage of grep, awk, sed, cut, and sort for log analysis and extracting information.

1.4 Automation Basics

Bash Scripting Fundamentals

Writing functional bash scripts utilizing variables, loops, conditionals, and functions.

1.5 System Operations

System Monitoring & Processes

Monitoring performance using top, ps, kill, and managing services effectively via systemd/systemctl.

Phase 2: Networking Deep Dive

Protocols & Analysis

+

TCP/IP, OSI model, subnetting, and analyzing traffic with Wireshark.

2.1 Network Models

OSI & TCP/IP Deep Dive

Deconstructing the layers of the OSI model and how they map to the practical TCP/IP suite.

2.2 Network Addressing

IP Addressing & Subnetting

Mastering IPv4/IPv6, CIDR notation, subnet masks, and routing principles.

2.3 Core Protocols

Understanding Network Services

Deep dive into DNS, HTTP, FTP, SSH, and SMB. Understanding packet structures and common ports.

2.4 Traffic Analysis

Packet Sniffing with Wireshark

Capturing and dissecting network traffic, utilizing display filters, and identifying malicious patterns.

2.5 Network Discovery

Active Mapping & Recon

Using tools like Nmap for host discovery, port scanning techniques, and OS footprinting.

Phase 3: Bash & Automation

Scripting

+

Writing robust scripts to automate reconnaissance and system tasks.

3.1 Advanced Scripting

Advanced Bash Techniques

Mastering arrays, string manipulation, and leveraging regular expressions within scripts.

3.2 Workflow Efficiency

Automating Recon Flows

Chaining multiple tools together to build an automated enumeration pipeline.

3.3 Reliability

Error Handling & Logging

Implementing trap commands, exit codes, and comprehensive logging mechanisms in bash.

3.4 Web Interactions

API Integration with cURL

Querying REST APIs, handling JSON responses with jq, and automating web requests.

3.5 Task Management

Cron Jobs & Scheduling

Automating script execution intervals and establishing persistence using cron and systemd timers.

Phase 4: Python for Offense

Tool Development

+

Building custom tools, socket programming, and interacting with APIs.

4.1 Programming Basics

Python for Hackers

Core language features, data structures, context managers, and setting up environments.

4.2 Data Processing

File I/O & Text Formats

Parsing raw text, JSON, and CSV data efficiently. Scraping basics with BeautifulSoup.

4.3 Low-Level Network

Network Socket Programming

Building custom TCP/UDP clients and servers. Writing a custom port scanner.

4.4 Web Interactions

Mastering the Requests Library

Handling sessions, custom headers, cookies, and automating login sequences.

4.5 Custom Exploits

Developing Exploitation Scripts

Translating proof-of-concepts into weaponized Python scripts and argument parsing.

Phase 5: Web Foundations (HTTP/JS)

The Language of Web

+

Understanding the protocol of the web, headers, cookies, and DOM manipulation.

5.1 Protocol Mechanics

HTTP Methods & Status Codes

Dissecting GET, POST, PUT, DELETE, and understanding server responses.

5.2 State Management

Headers, Cookies, & Sessions

Tracking state across a stateless protocol. Understanding JWTs and session tokens.

5.3 Client-Side Execution

Understanding the DOM

Exploring how browsers render HTML and manage the Document Object Model tree.

5.4 Scripting the Web

JavaScript Security Context

Variable scopes, closures, event listeners, and common secure coding pitfalls.

5.5 Browser Security Mechanisms

SOP & CORS

Mastering the Same-Origin Policy and Cross-Origin Resource Sharing rules.

Phase 6: Recon Methodology

Information Gathering

+

Passive and active information gathering, subdomain enumeration, and OSINT.

6.1 Open Source Intel

Passive OSINT Techniques

Leveraging WHOIS, reverse DNS, search engines, and archives (Wayback Machine).

6.2 Advanced Searching

Google Dorking for Hackers

Crafting advanced search queries to uncover exposed files and vulnerable endpoints.

6.3 Surface Area Mapping

Subdomain Enumeration

Combining passive (crt.sh, Amass) and active (Sublist3r, brute-forcing) discovery methods.

6.4 Content Discovery

Directory & File Fuzzing

Utilizing tools like ffuf and Gobuster with custom wordlists to locate hidden paths.

6.5 Fingerprinting

Identifying Technologies & WAFs

Detecting backend stacks (Wappalyzer, WhatWeb) and bypassing Web Application Firewalls.

Phase 7: Web Exploitation I

OWASP Top 10

+

SQL injection, XSS, and IDOR vulnerabilities.

7.1 Database Exploits

SQL Injection (SQLi)

Discovering and exploiting Error-based, Union-based, and Blind SQL injection flaws.

7.2 Client-Side Attacks

Cross-Site Scripting (XSS)

Hunting for Reflected, Stored, and DOM-based XSS to execute malicious scripts.

7.3 Access Control Flaws

Insecure Direct Object References

Manipulating parameters to access unauthorized documents or administrative actions.

7.4 Rogue Requests

Cross-Site Request Forgery (CSRF)

Forcing authenticated users to execute unwanted actions and bypassing CSRF tokens.

7.5 System Compromise

Command Injection

Escaping user input contexts to execute arbitrary OS commands on the target host.

Phase 8: Web Exploitation II

Advanced Attacks

+

Server-side attacks, deserialization, and template injection.

8.1 Internal Network Access

Server-Side Request Forgery

Pivoting through vulnerable web servers to access internal infrastructure and cloud metadata.

8.2 Entity Expansion

XML External Entity (XXE)

Abusing XML parsers to read local files, execute SSRF, and cause Denial of Service.

8.3 Object Abuse

Insecure Deserialization

Exploiting native serialization formats (Java, PHP, Python) to achieve Remote Code Execution.

8.4 Template Engine Exploits

Server-Side Template Injection

Escaping sandbox environments in Jinja, Twig, or FreeMarker to execute system commands.

8.5 Modern Web Abstraction

API Security & Exploitation

Analyzing REST/GraphQL endpoints, mass assignment, and lack of rate limiting.

Phase 9: Authentication Attacks

Identity & Access

+

Brute force, credential stuffing, OAuth abuse, and session management.

9.1 Credential Recovery

Password Cracking Techniques

Utilizing Hashcat and John the Ripper, rule-based mutations, and optimizing wordlists.

9.2 Logic Flaws

Bypassing Authentication Mechanisms

Exploiting weak reset password flows, default credentials, and parameter tampering.

9.3 Modern Auth Systems

Attacking OAuth 2.0 & SAML

Identifying misconfigured redirect URIs, stealing tokens, and manipulating signature processes.

9.4 Token Abuse

Session & JWT Manipulation

Alg-none attacks on JSON Web Tokens, key confusion routines, and session hijacking.

9.5 Secondary Defenses

MFA Bypass Strategies

Bypassing multi-factor authentication via race conditions, brute-forcing OTC, or social engineering.

Phase 10: Privilege Escalation (Linux)

Rooting the Box

+

Kernel exploits, SUID binaries, and misconfigured cron jobs.

10.1 System Profiling

Enumeration for Privilege Escalation

Automated enumeration using LinPEAS, evaluating running processes, and spotting weak file permissions.

10.2 Binary Abuses

Exploiting SUID/SGID & Capabilities

Leveraging misconfigured executable permissions, GTFOBins, and Linux capabilities for elevated access.

10.3 Background Tasks

Abusing Cron Jobs & Scheduled Tasks

Tinkering with writable scripts, ambiguous environmental paths, and wildcards in cron tasks.

10.4 System Core

Linux Kernel Exploitation

Identifying kernel versions, understanding Dirty COW (CVE-2016-5195), and compiling C-based exploits.

10.5 Environmental Quirks

Shared Libraries & LD_PRELOAD

Exploiting sudo rights on environment variables and manipulating shared object injections.

Phase 11: Windows Internals

OS Architecture

+

Understanding the registry, services, processes, and PowerShell.

11.1 Core Components

Exploring the Windows Registry

Understanding hive structures, autostart execution points (ASEPs), and querying SAM data.

11.2 System Operations

Managing Services and Processes

Investigating unquoted service paths, weak service permissions, and handle manipulation.

11.3 Security Boundaries

Access Tokens & UAC Mechanics

Deconstructing SeImpersonatePrivilege, token stealing, and User Account Control bypass vectors.

11.4 Administrative Shell

PowerShell for Offensive Action

Bypassing execution policies, AMSI evasion, and employing tools like PowerSploit.

11.5 Telemetry & Logs

Windows Event Logs & Sysmon

Identifying crucial log IDs, tracing attacker footprints, and understanding EDR detection vectors.

Phase 12: Active Directory Attacks

Enterprise Security

+

Kerberoasting, AS-REP roasting, mimicking, and lateral movement.

12.1 Domain Discovery

Active Directory Enumeration

Mapping out trust relationships and attack paths using BloodHound, PowerView, and ldapsearch.

12.2 Ticket Extraction

Kerberoasting & AS-REP Roasting

Requesting service tickets to crack offline and abusing users without pre-authentication.

12.3 Hash Abuse

Pass-the-Hash & Pass-the-Ticket

Authenticating across services using NTLM hashes or forged Kerberos tokens directly in memory.

12.4 Domain Dominance

DCSync & The Golden Ticket

Simulating domain controllers to extract the KRBTGT hash and forging virtually unlimited access tokens.

12.5 Host Hopping

Lateral Movement Capabilities

Executing remote processes via PsExec, WMI, SmbExec, and mapping administrative shares.

Phase 13: Post-Exploitation Ops

Persistence & Looting

+

Persistence mechanisms, data exfiltration, and covering tracks.

13.1 Retaining Access

Establishing Deep Persistence

Injecting backdoors, scheduled tasks, WMI event subscriptions, and hidden registries.

13.2 Data Extraction

Looting & Secure Exfiltration

Archiving sensitive data, evading DLP filters, and smuggling data out via DNS or ICMP.

13.3 Built-in Capabilities

Living off the Land (LotL)

Conducting post-exploitation purely through native OS binaries (Certutil, MSBuild, BITSAdmin) to avoid detection.

13.4 Network Traversal

Pivoting and Port Forwarding

Routing attack traffic into subnets via SOCKS proxies, Chisel, Proxychains, and SSH tunneling.

13.5 Anti-Forensics

Clearing Tracks & Evasion

Wiping shell histories, selectively removing event logs, and timestamp modification (timestomping).

Phase 14: Binary Exploitation Basics

Memory Corruption

+

Stack-based buffer overflows, shellcoding, and basic reverse engineering.

14.1 Foundation

Introduction to Assembly Language

Understanding x86 and x64 CPU architectures, registers, and basic instruction sets.

14.2 Code Analysis

Reverse Engineering Fundamentals

Decompiling and dissecting ELF and PE binaries using Ghidra, IDA Pro, and Radare2.

14.3 Spatial Control

Stack-Based Buffer Overflows

Calculating offsets, overwriting instruction pointers (EIP/RIP), and executing arbitrary data.

14.4 Injected Payloads

Writing Custom Shellcode

Developing platform-specific bytecodes to spawn shells while avoiding bad characters.

14.5 Defeating Defenses

Return-Oriented Programming (ROP)

Bypassing modern mitigations like DEP and ASLR by chaining existing binary gadgets.

Phase 15: Red Team Operations

Adversary Simulation

+

C2 infrastructure, evasion techniques, and full-scope engagement simulation.

15.1 Command Centers

Setting up C2 Infrastructure

Deploying tools like Cobalt Strike, Mythic, or Sliver via redirectors and obfuscated domains.

15.2 Initial Access

Advanced Phishing Campaigns

Crafting persuasive payloads (macros, ISOs, JS) and setting up tracking mechanisms.

15.3 Stealth & OPSEC

Operational Security Considerations

Maintaining burner infrastructures, segmenting tools, and adopting behaviors to remain undiscovered.

15.4 The Human Element

Physical Security & Social Engineering

Tailgating, dropping malicious USBs, and manipulating targets over vishing campaigns.

15.5 Debriefing

Reporting & Remediation Consultation

Translating complex attack kill-chains into clear, executive-friendly reports with actionable fixes.
